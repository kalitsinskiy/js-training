**Evaluation Format:**
- Total questions: 10
- Students: 5 candidates
- Each student gets: 2 questions randomly
- Prepare answers for ALL 10 questions!

---

**Q1:** What is the Singleton pattern? When is it useful and when is it harmful? Implement one in JS.

**Q2:** Explain the difference between Factory and Constructor. When would you choose Factory?

**Q3:** What is the Observer pattern? How does it differ from Pub/Sub? Give a real-world example.

**Q4:** Explain the Strategy pattern. How does it help with the Open/Closed Principle?

**Q5:** What is the Decorator pattern? How is it different from inheritance? Show a JS example.

**Q6:** What is the Proxy pattern? Name 3 real use cases (logging, caching, validation, etc.).

**Q7:** What is the Module pattern in JavaScript? What problem does it solve?

**Q8:** Explain the Builder pattern. When is it better than a constructor with many arguments?

**Q9:** What is the Adapter pattern? Give an example where you would use it in a real project.

**Q10:** What is the Command pattern? How does it enable undo/redo functionality?

---

**Evaluation Criteria:**
- Can name the pattern category (creational/structural/behavioral)
- Can explain the problem it solves, not just describe the solution
- Can write a simplified JS implementation
- Knows real-world use cases
