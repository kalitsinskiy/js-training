2. content-box vs border-box

content-box (default)

200px width

3. Margin Collapse

Adjacent vertical margins collapse into a single margin (the larger one wins):

margin-bottom: 30px
margin-top: 20px

The gap between the two boxes is 30px (not 50px). The larger margin wins — they collapse.
