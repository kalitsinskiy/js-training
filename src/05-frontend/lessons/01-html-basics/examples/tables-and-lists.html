6. Definition List — Secret Santa Glossary

Room
A group of people participating in a gift exchange. Each room has a unique code.
Draw
The random assignment of gift givers to recipients. Each person is assigned exactly one other person.
Wishlist
A list of gift ideas that a participant would like to receive. Helps the gift giver choose something appropriate.
Budget
The maximum amount of money to spend on a gift, agreed upon by all room members.
Room Code
A short alphanumeric string (e.g., ABC123) used to invite others to join a room.
