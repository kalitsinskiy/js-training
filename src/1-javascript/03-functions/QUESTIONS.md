# Week 3: Functions - Evaluation Questions

**Evaluation Format:**
- Total questions: 10
- Students: 5 candidates
- Each student gets: 2 questions randomly
- Prepare answers for ALL 10 questions!

---

**Q1:** What are the three ways to define functions in JavaScript? Explain the differences between them.

**Q2:** What is hoisting? How does it affect function declarations vs function expressions?

**Q3:** Explain arrow functions. What are their benefits and limitations compared to regular functions?

**Q4:** What is the difference in `this` binding between arrow functions and regular functions? Provide an example.

**Q5:** What is a higher-order function? Give an example of creating and using one.

**Q6:** Explain what a callback function is. Why are callbacks important in JavaScript?

**Q7:** What is the difference between parameters and arguments? Explain rest parameters (`...args`).

**Q8:** How does implicit return work in arrow functions? When can you use it and when must you use explicit return?

**Q9:** What is function scope? Can inner functions access variables from outer functions? Explain with an example.

**Q10:** Explain the concept of pure functions. What makes a function "pure" and why is it important?

---

**Evaluation Criteria:**
- Clear explanation of concepts
- Ability to provide code examples
- Understanding of "why", not just "what"
- Real-world use cases
