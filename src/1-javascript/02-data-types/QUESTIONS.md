# Week 2: Data Types - Evaluation Questions

**Evaluation Format:**
- Total questions: 10
- Students: 5 candidates
- Each student gets: 2 questions randomly
- Prepare answers for ALL 10 questions!

---

**Q1:** What are the 7 primitive data types in JavaScript? Provide an example of each.

**Q2:** Explain the difference between `null` and `undefined`. When would you use each?

**Q3:** What is type coercion? Give examples of implicit and explicit type conversion.

**Q4:** What are the 6 falsy values in JavaScript? Why is understanding them important?

**Q5:** Why does `typeof null` return `"object"`? Is `null` actually an object?

**Q6:** Explain the difference between primitive types and reference types. How are they compared?

**Q7:** What happens when you use the `+` operator with different types? Give examples with strings, numbers, and booleans.

**Q8:** What is `NaN`? How do you check if a value is `NaN`? Why doesn't `NaN === NaN` work?

**Q9:** Explain type conversion in the following expressions:
```javascript
"5" - 3    // ?
"5" + 3    // ?
!!"hello"  // ?
```

**Q10:** What's the difference between `==` and `===`? Which one should you use and why?

---

**Evaluation Criteria:**
- Clear explanation of concepts
- Ability to provide code examples
- Understanding of "why", not just "what"
- Real-world use cases
