# Week 11: Closures - Evaluation Questions

**Evaluation Format:**
- Total questions: 10
- Students: 5 candidates
- Each student gets: 2 questions randomly
- Prepare answers for ALL 10 questions!

---

**Q1:** What is a closure? Explain in your own words with a simple code example.

**Q2:** What is lexical scope? How does it differ from dynamic scope?

**Q3:** What does it mean that "closures capture variables by reference, not by value"? Show a code example demonstrating this.

**Q4:** Explain the classic "loop closure" problem with `var`. Why does it happen? How do you fix it?

**Q5:** What are factory functions? How do closures enable them?

**Q6:** What is memoization? Implement a `memoize` function using closures.

**Q7:** What is partial application? How does it differ from currying?

**Q8:** How do closures enable the "module pattern"? Why was this important before ES6 modules?

**Q9:** What is the difference between a closure and a class for creating objects with private state? When would you choose each?

**Q10:** Can closures cause memory leaks? When and why?

---

**Evaluation Criteria:**
- Clear explanation of concepts
- Ability to provide code examples
- Understanding of "why", not just "what"
- Real-world use cases
